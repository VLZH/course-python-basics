## HTML

**HTML** - стандартизированный язык разметки документов во Всемирной паутине. Большинство веб-страниц содержат описание разметки на языке HTML (или XHTML). Язык HTML интерпретируется браузерами; полученный в результате интерпретации форматированный текст отображается на экране монитора компьютера или мобильного устройства.

Изначально язык разрабатывался для хранения документов и научной документации, подразумевалось, что пользоваться им может любой человек. Для описания документов используются специальные последовательности символов(дескрипторы) - теги. С помощью тегов вы описываете документ, выглядят они примерно так: `<p/>`.

Пример:

```html
<p>I am parahraph</p>
<p>I am second parahraph</p>
```

:::info
**Дескри́птор** - лексическая единица (слово, словосочетание) информационно-поискового языка, служащая для описания основного смыслового содержания документа или формулировки запроса при поиске документа (информации) в информационно-поисковой системе.
:::

  Строчные элементы — это те, которые содержатся в элементах блочного уровня и окружают только малые части содержимого документа, не целые абзацы и группировки контента. Строчные элементы не приводят к появлению новой строки в документе: они обычно встречаются внутри абзаца текста, например, элемент `<a>` (ссылка) или акцентирующие элементы вроде `<em>` или `<strong>`.

  ```html
  <p>четвертый</p>
  <p>пятый</p>
  <p>шестой</p>
  ```

### Атрибуты

У элементов также могут быть атрибуты, которые выглядят так:

```html
<p id="first_paragraph" class="paragraph">...</p>
```

Атрибуты содержат дополнительную информацию об элементе, которая, по вашему мнению, не должна отображаться в содержимом элемента. В данном случае атрибут `class` позволяет вам дать элементу идентификационное имя, которое в дальнейшем может быть использовано для обращения к элементу с информацией о стиле и прочими вещами.

Атрибут должен иметь:

1. Пробел между атрибутом и именем элемента (или предыдущим атрибутом, если у элемента уже есть один или несколько атрибутов).
2. Имя атрибута и следующий за ним знак равенства.
3. Значение атрибута, заключенное в кавычки.

### Основные теги

- `<html/>` - Обозначает html-документ
- `<head/>` - Обозначает блок мета-информации о документе
- `<body/>` - Обозначает блок содержиимого

- `<p/>` - Параграф
- `<h1/>, <h2/>, <h3/>, <h4/>, <h5/>, <h6/>` - Заголовок 1-6 уровня
- `ul` - Неупорядоченный список
- `ol` - Упорядоченный список
- `li` - Упорядоченный список

### Структура HTML-документа

На текущий момент мы говорили только про элементы на странице, но надо также обозначить как размечается html-документ.

```html
<!DOCTYPE html>
<!-- Обозрачаем что это html-документ -->
<html lang="en">
  <!-- обозначаем html содержимое -->
  <head>
    <!-- заголовочная информация -->
    <meta charset="UTF-8" />
    <!-- кодировка документа -->
    <title>Page title</title>
    <!-- название документа -->
  </head>
  <body>
    <!-- тело документа -->
    <p>
      Sit accusamus corporis error unde dolor distinctio? Enim facere alias?
    </p>
  </body>
</html>
```
