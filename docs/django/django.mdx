## Состав фреймворка

- `django.core` - Базовые пакеты отвечающие за сигналы, работу с файлами и.т.п
- `django.apps` - Хранит и предоставляет информацию о приложениях (`applications`)
- `django.conf` - Пакет для работы с конфигурацией проекта
- `django.contrib.*` - Приложения входящие в состав фреймворка
- `django.db` - Пакеты для работы с базами данных
- `django.http` - Пакет отвечающий за... как понятно из названия `HTTP` протокол.
- `django.view` - Вспомогательные функции для обработчиков запроса
