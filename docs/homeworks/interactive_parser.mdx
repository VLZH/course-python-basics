Мы с вами уже реализовали небольшой проект состоящий из 2 частей:
- Серверный код на `Flask`
- Клиентский код на `JS`, который позволяет нам отправлять `AJAX` запросы на адрес `/...?`

В ходе изучения мы с вами реализовали ряд view-функций, при запросе на которые мы:
- Находили фильмы по их названиям
- Генерировали html из markdown
- Парсили Open-Graph разметку страницы в интернете
