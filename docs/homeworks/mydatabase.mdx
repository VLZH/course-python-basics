###  Домашнее задаине создано что бы проверки и закрепления ваших знаний по работе с базой данных 

Сервис предоставляет возможность добавлять объявления пользователям. Вот примеры того что представляют собой сущности:

**Пользователь**:
  - Имя
  - Фамилия
  - Username
  - Password

**Объявление**:
  - Название объявления
  - Описание объявление
  - Короткое описание
  - Изображения
  - Ссылка на автора (таблица пользователей)
  - Ссылка на категории (таблица категорий)
    
**Категории**
  - Название категорий


#### Что вам предстоит сделать?
- Создать базу данных в Postgresql
- Создать таблицы в базе данных
- Создать несколько записей в этих таблицах
- Создать SQL файл где сохраняем все написанные запросы в базу данных

### Подсказки 
Docker
```bash
docker ps #список всех запущенных контейнеров
docker ps -a #список всех контейноров
docker exec -it container_id bash #зайти в докер контейнер
psql user=postgres #для входа в postgresql внутри контейнера
```
