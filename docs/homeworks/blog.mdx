### Что вам предстоит сделать?
Вы должны написать функционал который позволить редактировать, удалять и создавать категорий и авторов 
как это было сделано с постами 

### Шаги 
Вам нужно сверстать формы и страницы
- Сверстать форму для создания и редактирования категорий
- Сверстать страницу для удаления для категорий 
- Сверстать форму для регистрации авторов
- Сверстать страницу для удаления для авторов
Вам нужно добавить сервисы  AuthorsDTO и CateoryDTO с функциями
- Для создания новой записи в базе данных
- Для обновления записей в базе данных
- Для удаления записей из базы данных
Вам нужно добавить функции обработчики и роуты для этих функционалов
