#### Вам предстоит интересная задача  вам нужно написать функцию которая по номеру месяца какой сезон
### Какие шаги вам нужно для этого сделать?
- Обьявить функцию season()
- Первая строка функции печатает 'Добро пожаловать в функцию 'Какой сейчас сезон?''
- На второй строке мы обьевляем переменную month которая с помощью input() попросит ввести  число от 1 до 12
- Далее мы пишем условие if..else которая проверит значение month и выведит разные сообщения 

### Подсказки
- Как обьявить функцию?
функцию обьявляют с помощию ключевого слова def
```python
def myfunction():
    pass
```
Не забудь в конце поставить ':' 
- Как напечатать что то в терминален?
Это мы делаем с помощью print()
```python
print('my function')
```  
- Как работает input()
```python
name = int(input('Ваше имя:'))
```
input позовляет взять у пользователя какое либо значение через терминал

- Как использовать if..else
if..else позволяет провеять что то и взависимости от результата что то делать
```python
number = int(input('Введите число от 1 до 4:'))
if number == 1 or number == 2:
    print('Вы ввели 1 или 2')
if number == 3 or number == 4:
    print('Вы ввели 3 или 4')
else:
    print('число должно быть 1,2,3 или 4')
```
