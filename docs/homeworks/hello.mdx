### Переменные 
Переменная это имя которое вы даете чему то в python и оно хранит в себе какую то информацию
далее будут комментарии в python  комментарии пишет через #.Комментарии  пишуться в коде что бы другим разработчикам  
легче было понять ваш код

```python
number = 7 #Теперь number это переменная со значением 7 попробуем вывести его в консоль
print(number)
number2 = 8
print(number2)
print(number+number2)
```
Input  это модуль который позволяет запросить у ползьвателя прямо в терминале какое то значением
```python
number=input('число:')#здесь он в терминале попросит вас ввести код терминал это окно с права вам нужно будет вписать туда число
print(number)   
```
