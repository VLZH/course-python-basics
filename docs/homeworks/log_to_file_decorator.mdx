---
title: Декоратор log
id: log_to_file_decorator
---

Вам надо написать конфигурируеммый декоратор `log`, который логгирует в файл все вызовы функции к которой он был применен. Как уже сказано, декоратор конфигурируемый, а значит применение выглядит примерно так:

```python
@log("./my_func.log")
def my_func():
    pass

my_func()
my_func()
```
