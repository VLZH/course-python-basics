Как и всегда первое что мы с вами сделаем изучая новую технологию это - Hello World проект.

Давайте воспользуемся модулем `venv` и создадим виртуальное окружение для нашего проекта:
```bash
mkdir -p ~/projects/flask_hello_world # создаем папку для проекта
cd ~/projects/flask_hello_world # и переходим в нее
python3 -m venv ./.pyenv # создаем виртуальное окружение
source .pyenv/bin/activate # активируем его
```
